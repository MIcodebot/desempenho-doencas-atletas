# 🏃 desempenho-doencas-atletas

Análise de dados sobre o **impacto de doenças no desempenho de atletas**, desenvolvida com Python e Pandas. Explora padrões entre condições de saúde e resultados esportivos.

---

## 📋 Sobre o Projeto

Este repositório contém um projeto de **análise de dados** que investiga como diferentes doenças afetam o desempenho de atletas. O projeto faz parte dos estudos em Ciência de Dados, aplicando técnicas de análise e visualização com Python.

---

## 🚀 Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/MIcodebot/desempenho-doencas-atletas.git
   ```
2. Instale as dependências:
   ```bash
   pip install pandas matplotlib
   ```
3. Execute a análise:
   ```bash
   python analise.py
   ```

## 📊 Objetivos da Análise

- Identificar quais doenças mais afetam atletas de alto rendimento

- Visualizar dados com gráficos informativos
- Extrair insights sobre saúde esportiva
